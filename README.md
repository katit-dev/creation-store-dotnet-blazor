\# Creation Store - ASP.NET Core Web API + Blazor



Creation Store is a digital product e-commerce web application built with ASP.NET Core Web API, Blazor, SQL Server, and Entity Framework Core.



\## Technologies



\- ASP.NET Core Web API

\- Blazor WebAssembly

\- SQL Server

\- Entity Framework Core

\- JWT Authentication

\- Swagger



\## Main Features



\- User login/register

\- Role-based authorization

\- Product listing

\- Product filtering

\- Cart management

\- Checkout order

\- Order history

\- Admin product management
